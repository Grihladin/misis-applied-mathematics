# Systems Analysis Coursework

Six Python exercises from the Systems Analysis course at MISIS:

1. Read tabular data and represent hierarchical structures as trees.
2. Traverse, search, serialize, and display tree structures.
3. Calculate management relationships and structural entropy in a hierarchy.
4. Calculate probability distributions, entropy, and information measures for
   dice sums and products.
5. Compare rankings and identify contradictory preference pairs.
6. Estimate agreement between expert rankings using a Kendall-style
   coefficient.

The repository is preserved as historical coursework written manually without
generative-AI-produced code. The implementations are educational exercises, not
a maintained library.
