# Mathematical Statistics Laboratory Work

A collection of exploratory Python exercises completed for the Mathematical
Statistics course at MISIS in 2023.

## What was explored

- `TASK_1`: ARIMA time-series modelling, likelihood estimation, parameter
  inspection, Metropolis-Hastings experimentation, and train/test evaluation.
- `TASK_2`: log returns, kernel regression, k-nearest-neighbour regression,
  comparisons with linear regression, and kernel density estimation.
- `TASK_3`: mixtures of normal distributions, normality testing, likelihood
  calculations, Gaussian mixture modelling, and Metropolis-Hastings sampling.

## Technologies

- Python
- NumPy and pandas
- SciPy and statsmodels
- scikit-learn
- Matplotlib

## Running the exercises

Install the packages used across the collection:

```bash
pip install numpy pandas scipy statsmodels scikit-learn matplotlib
```

Each Python file is a standalone experiment and can be run directly, for
example:

```bash
python TASK_1/2.py
python TASK_2/3.py
python TASK_3/5.py
```

## Known limitations

- These files are preserved close to their submitted state and are not a
  cohesive software package.
- Some exercises are incomplete, contain placeholder files, or require
  mathematical and correctness review before reuse.
- Several filenames, comments, plot labels, and notes remain in Russian.
- Randomized examples do not consistently set seeds.
- Dependencies were not originally pinned.

## Historical note

The exercises were written by hand before generative AI coding assistants
became widely available. They are retained to show the original learning
process, including unfinished and exploratory work.
