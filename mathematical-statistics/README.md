# Mathematical Statistics Laboratory Work

A collection of Python exercises from the Mathematical Statistics course at
MISIS.

## Topics

- `TASK_1`: ARIMA time-series modelling, likelihood estimation, parameter
  inspection, Metropolis-Hastings experimentation, and train/test evaluation.
- `TASK_2`: log returns, kernel regression, k-nearest-neighbour regression,
  comparisons with linear regression, and kernel density estimation.
- `TASK_3`: mixtures of normal distributions, normality testing, likelihood
  calculations, Gaussian mixture modelling, and Metropolis-Hastings sampling.

The files are preserved as submitted coursework. Several filenames, comments,
plot labels, and short exercise notes remain in Russian; this README provides
the concise English guide.

This is historical student work written manually without generative-AI-produced
code. Some scripts are exploratory and may need dependency or correctness
updates before reuse.
